📄 Project Description

This project is a responsive college event management website developed using HTML, CSS, and JavaScript. The application is designed to simulate a real-world event platform where users can explore different college events, view detailed information, and register for participation.

The website consists of three main pages: Home, Events, and Registration, providing a structured and user-friendly interface.

🔹 Features & Functionality
🏠 Home Page
Provides an overview of the event platform
Welcomes users with an engaging UI
Includes navigation to explore events easily
🎭 Events Page
Displays six different types of events:
Dance
Singing
Drama
Band
Fashion Show
Talent Show
Each event is presented using interactive cards
Every card contains:
Event title
Brief description
“Details” button → redirects to a dedicated event page
“Register” button → redirects to registration form
📄 Event Detail Pages
Separate pages for each event
Provide detailed information about the selected event
Improve user understanding before registration
📝 Registration Page
Dynamic registration form built using JavaScript
Supports:
Input validation (name, email, phone, etc.)
Dynamic fields (for solo/group participation)
Enhances user experience through real-time interaction
⚙️ Technologies Used
HTML → Structure of the website
CSS → Styling, layout, and responsiveness
JavaScript → Interactivity, DOM manipulation, and validation
